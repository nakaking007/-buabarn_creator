Buabarn Local Video Maker
============================================================

ระบบนี้คืออะไร
------------------------------------------------------------
ระบบเสริมสำหรับ Buabarn VIP Creater Tools
ใช้สร้างวิดีโอแนวตั้ง MP4 จากภาพนิ่ง 1 ภาพ และเสียงหรือสคริปต์บรรยาย
เหมาะกับ TikTok, Reels, YouTube Shorts, คลิปขายสินค้า และคลิปเล่าเรื่องสั้น

ทำงานผ่านอะไร
------------------------------------------------------------
- Python
- ffmpeg.exe ที่อยู่ในโฟลเดอร์นี้
- gTTS เฉพาะกรณีที่ต้องการสร้างเสียงไทยอัตโนมัติผ่าน Google TTS

ไฟล์สำคัญ
------------------------------------------------------------
- run_local_video.bat: ดับเบิลคลิกเพื่อสร้างวิดีโอ
- local_video_maker.py: สคริปต์หลัก
- config.json: ตั้งค่าภาพ เสียง ขนาดวิดีโอ และไฟล์ส่งออก
- voice_script.txt: ข้อความบรรยาย
- input/product.jpg: ภาพต้นฉบับ
- output/buabarn_local_video.mp4: ไฟล์วิดีโอที่สร้างเสร็จ

วิธีใช้แบบง่าย
------------------------------------------------------------
1. วางภาพที่ต้องการในโฟลเดอร์ input
2. แก้ชื่อภาพใน config.json ให้ตรงกับไฟล์ เช่น input/product.jpg
3. ถ้าต้องการใช้เสียงที่มีอยู่แล้ว ให้วางไฟล์เสียง mp3/wav และใส่ path ใน audio_file
4. ถ้าไม่ใส่เสียง ระบบจะสร้างวิดีโอจากภาพอย่างเดียว
5. ดับเบิลคลิก run_local_video.bat
6. ดูไฟล์ mp4 ในโฟลเดอร์ output

การใช้เสียงบรรยายด้วย gTTS
------------------------------------------------------------
ถ้าต้องการให้ระบบอ่านข้อความจาก voice_script.txt เป็นเสียงไทย:
1. เปิด command prompt ในโฟลเดอร์นี้
2. รันคำสั่ง python -m pip install gTTS
3. เปิด config.json
4. เปลี่ยน "use_gtts": false เป็น "use_gtts": true
5. ดับเบิลคลิก run_local_video.bat

ข้อจำกัด
------------------------------------------------------------
- นี่ไม่ใช่ AI สร้างวิดีโอจากศูนย์ แต่เป็นระบบประกอบภาพนิ่ง + เสียง เป็นวิดีโอจริง
- ถ้าใช้ gTTS ต้องมีอินเทอร์เน็ต
- ถ้าต้องการวิดีโอเคลื่อนไหวแบบ AI จริง ยังต้องใช้ API วิดีโอภายนอก
- เหมาะสำหรับเพิ่มความน่าเชื่อถือของระบบ เพราะสร้างไฟล์ MP4 ได้จริงแบบ local

คำแนะนำการใช้ร่วมกับระบบใหญ่
------------------------------------------------------------
1. ใช้ Buabarn VIP Creater Tools สร้าง Voice Over, Caption และ Storyboard
2. คัดลอก Voice Over มาใส่ voice_script.txt
3. วางภาพสินค้า/ภาพ AI ใน input
4. ใช้ Buabarn Local Video Maker สร้าง mp4
5. นำ mp4 ไปลง TikTok, Reels หรือ YouTube Shorts
