ระบบนี้เป็น Chrome Extension สำหรับช่วยสร้าง prompt และชิ้นงานคอนเทนต์ AI

ข้อมูลที่ระบบอาจจัดเก็บ:

- API/Token ที่ผู้ใช้กรอกสำหรับสร้างภาพหรือวิดีโอ
- การตั้งค่าภายใน extension
- ข้อความ prompt หรือชิ้นงานที่ผู้ใช้สร้างในเครื่อง

ข้อมูลเหล่านี้จัดเก็บในเครื่องของผู้ใช้ผ่าน Chrome storage
ระบบไม่ตั้งใจส่งข้อมูลส่วนตัวไปยังผู้พัฒนา

เมื่อผู้ใช้กดสร้างภาพหรือวิดีโอ ระบบจะส่ง prompt และ API/Token ไปยังผู้ให้บริการภายนอกที่ผู้ใช้เลือก เช่น Hugging Face
ผู้ใช้ควรอ่านเงื่อนไขและนโยบายของผู้ให้บริการ API นั้น ๆ ก่อนใช้งาน

ระบบมีปุ่มอ่านชื่อ/คำอธิบายจากแท็บเว็บ เพื่อช่วยดึงข้อมูลหน้าเว็บที่ผู้ใช้เปิดอยู่
บางเว็บไซต์อาจไม่อนุญาตให้ดึงข้อมูลได้

ผู้ใช้เป็นผู้รับผิดชอบ API/Token ของตนเอง
ไม่ควรส่ง API/Token ให้ผู้อื่นหากไม่จำเป็น
