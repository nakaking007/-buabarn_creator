BUABARN VIP - Web Demo Deploy

โฟลเดอร์นี้คือเว็บตัวอย่างแบบ static สำหรับเปิดดูหรืออัปโหลดขึ้นโฮสต์ได้ทันที

ไฟล์หลัก:
- index.html = หน้าเว็บหลัก
- styles.css = รูปแบบหน้าเว็บ
- *.js = ระบบสร้าง output และเชื่อม API
- assets/ = ภาพประกอบแต่ละหมวด
- readme.txt = คู่มือใช้งาน
- deploy-guide.txt = คู่มือติดตั้ง/ตรวจระบบ

วิธีเปิดในเครื่อง:
1. ดับเบิลคลิก index.html
2. หรือเปิดผ่าน local server เพื่อทดสอบเหมือนเว็บจริง

วิธี deploy แบบง่าย:
1. เข้า Netlify Drop: https://app.netlify.com/drop
2. ลากทั้งโฟลเดอร์ web_demo_BUABARN ไปวาง
3. รอระบบสร้างลิงก์เว็บ

หมายเหตุ:
- โหมดเว็บตัวอย่างอ่านแท็บเว็บอื่นไม่ได้ เพราะไม่ใช่ Chrome Extension
- ปุ่มสร้างภาพ/วิดีโอจริงต้องกรอก API ที่ใช้งานได้
- ถ้าจะใช้ฟังก์ชันอ่านข้อมูลหน้าเว็บ ต้องติดตั้งเป็น Chrome Extension
