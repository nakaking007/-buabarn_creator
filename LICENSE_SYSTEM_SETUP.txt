Buabarn VIP - วิธีเปิดระบบสิทธิ์รายเดือน / ตลอดชีพ

ไฟล์ที่เตรียมไว้ให้แล้ว:
- license-config.js = ตั้งค่าเปิด/ปิดระบบสิทธิ์ในหน้าเว็บ
- license-gate.js = หน้าตรวจ License Key
- LICENSE_BACKEND_GOOGLE_APPS_SCRIPT.js = โค้ด backend สำหรับ Google Apps Script
- license-customers-template.csv = ตารางตัวอย่างลูกค้า
- license-admin.html = ตัวช่วยสร้าง License Key และแถวข้อมูลลูกค้า

แพ็กเกจ:
1. รายเดือน 399 บาท
   - ใช้ได้ 30 วัน
   - วันหมดอายุอยู่ในคอลัมน์ expiresAt

2. ตลอดชีพ 1,999 บาท
   - plan = lifetime
   - expiresAt เว้นว่าง

ข้อมูลติดต่อ:
- ติดต่อ ดร.เขมวันต์
- กรุณาส่งสลิปการโอนมาที่
- ID Line: charnb015
- e-mail: charnb020@gmail.com

============================================================
ขั้นตอนทำ Google Sheet
============================================================

1. เปิด Google Drive
2. สร้าง Google Sheet ใหม่ชื่อ:
   Buabarn License

3. เปลี่ยนชื่อชีตด้านล่างให้เป็น:
   licenses

4. ใส่หัวตารางแถวแรก:
   licenseKey | customerName | lineId | email | plan | startsAt | expiresAt | active | note

5. เปิดไฟล์ license-customers-template.csv
   แล้วคัดลอกข้อมูลตัวอย่างไปวางใน Google Sheet ได้

============================================================
ขั้นตอนทำ Google Apps Script
============================================================

1. ใน Google Sheet กด Extensions
2. เลือก Apps Script
3. ลบโค้ดเดิม
4. เปิดไฟล์ LICENSE_BACKEND_GOOGLE_APPS_SCRIPT.js
5. คัดลอกโค้ดทั้งหมดไปวาง
6. กด Save

============================================================
ขั้นตอน Deploy Apps Script
============================================================

1. กด Deploy
2. เลือก New deployment
3. เลือก Type เป็น Web app
4. Execute as: Me
5. Who has access: Anyone
6. กด Deploy
7. คัดลอก Web app URL เก็บไว้

============================================================
เชื่อม URL เข้าหน้าเว็บ
============================================================

เปิดไฟล์ license-config.js แล้วแก้:

enabled: true,
validationUrl: 'วาง Web app URL ที่ได้จาก Google Apps Script'

ตัวอย่าง:

window.BUABARN_LICENSE_CONFIG = {
    enabled: true,
    validationUrl: 'https://script.google.com/macros/s/xxxxx/exec',
    ...
};

หลังแก้แล้ว ให้อัปโหลดไฟล์เหล่านี้ขึ้น GitHub Pages อีกครั้ง:
- index.html
- styles.css
- license-config.js
- license-gate.js

============================================================
วิธีเพิ่มลูกค้ารายเดือน
============================================================

1. เปิด license-admin.html
2. กรอกชื่อลูกค้า / Line / e-mail
3. เลือก รายเดือน 399 บาท / 30 วัน
4. กด สร้าง License Row
5. Copy แถวที่ได้
6. วางใน Google Sheet

ระบบจะตรวจ expiresAt:
- ถ้าวันหมดอายุยังไม่ถึง = เข้าใช้งานได้
- ถ้าวันหมดอายุผ่านแล้ว = เข้าไม่ได้

============================================================
วิธีเพิ่มลูกค้าตลอดชีพ
============================================================

1. เปิด license-admin.html
2. เลือก ตลอดชีพ 1,999 บาท
3. กด สร้าง License Row
4. วางใน Google Sheet

ระบบจะตรวจ plan = lifetime:
- ไม่มีวันหมดอายุ
- ใช้ได้ตลอดตราบใดที่ active = TRUE

============================================================
วิธีปิดสิทธิ์ลูกค้า
============================================================

เปลี่ยนคอลัมน์ active เป็น:
FALSE

============================================================
ข้อจำกัด
============================================================

- GitHub Pages เป็นเว็บ static จึงต้องมี Google Apps Script เป็น backend ตรวจสิทธิ์
- ห้ามใส่รหัสลับหรือข้อมูลสำคัญไว้ใน JavaScript หน้าเว็บ
- ระบบนี้เหมาะกับการขายเริ่มต้นและตรวจสิทธิ์ผ่าน License Key
